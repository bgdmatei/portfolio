PROJECT OVERVIEW: Fully-animated RPG created in Rails to make learning the basics of Ruby more fun and exciting. Features working combat, equipment, leveling/experience, and leaderboard systems. CONCEPT: During my time as a student at Le Wagon, I would often practice multiple-choice flashcards while not at school in order to reinforce what I learned. Although helpful, flashcards are not exactly the most exciting activity. Thus, after a brief discussion with two classmates, Code Wizard was born - a web app which gamifies multiple choice flashcards. CHALLENGES & SOLUTIONS: The biggest challenge in creating Code Wizard was ensuring the battle logic and associated animations worked as intended. This was difficult because we wanted to ensure data and the view was being updated without refreshing the page. To overcome this, we used JavaScript to render partials for each of the core "pages" (leaderboard, shop, profile) and the question logic (advance to next question or repeat question), as well as used gifs to animate the movement and attack actions.
